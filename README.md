##Remark
・US配列mac, US配列win, JIS配列mac, JIS配列winに対応  
・JIS配列で認識中はUS配列と同じキーを送信  

##既知の不具合
・JISモード時の;:'"が長押しに対応しない  
・"を連続で入力できない  
・US配列win時、ワンショットでIME切り替えができない(US配列には変換、無変換キーが無い為)  
・JISモードに切り替えた時PCBから変な音がする
